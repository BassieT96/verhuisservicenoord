# Verhuisservice Noord

Next.js App Router project met TypeScript, TailwindCSS en ESLint.

## Vereisten

- Node.js 20+
- npm 10+

## Installeren

```bash
npm install
```

## Development starten

```bash
npm run dev
```

App draait standaard op [http://localhost:3000](http://localhost:3000).

## Productie build maken

```bash
npm run build
```

## Productie server starten

```bash
npm run start
```
